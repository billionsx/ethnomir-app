## Что изменено по сравнению с v3.2

1. **Дата** изменена с 07.05.2026 на **04 мая 2026 г.** Номер договора синхронно изменён на **№ 04-05/2026** во всех 7 местах.
2. **Подписи** в разделе 17 и в форме Акта — теперь содержат **полные ФИО** Исполнителя и Заказчика как в реквизитах (раньше были сокращения «Иванов Е. В.» / «Чернова С. А.»).
3. Каждое **Приложение** (№ 1, № 2, № 3) теперь начинается **с новой страницы** — добавлены явные разрывы страницы.
4. **Сверка с справочником продукта** — обновлены формулировки без увеличения объёма работ:
   - предмет договора уточнён: указано, что приложение функционально замещает PMS отелей, POS ресторанов, CRM, маркетплейс услуг, админ-панели контента и инвестиционной недвижимости в едином контуре;
   - в M03 «Услуги» явно упомянут **трансфер**;
   - в реестре CRM модуль «Analytics» дополнен **RFM-сегментацией**;
   - в M00 «Главная» уточнено, что **промо-баннеры** ведут переходом на конкретные экраны приложения;
   - в правилах геймификации явно добавлено **сгорание баллов по сроку давности** (управляется через CRM);
   - убрано прямое имя внутренней таблицы (`crm_partner_access`) в 4.6 — заменено на функциональное описание партнёрского доступа.
