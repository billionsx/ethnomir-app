# Branch: `contracts`

Эта ветка содержит **правовые и договорные документы** проекта `ethnomir-app`.

Она намеренно оторвана от `main` (код приложения) и от `spravochnik-v2` (справочник продукта). Это — отдельное «юридическое измерение» репозитория: сюда складываются договоры, соглашения, акты и сопутствующая правовая документация.

## Структура

```
contracts/
└── ethnomir-app/      ← рабочий договор по разработке приложения
    ├── README.md
    ├── Договор_ethnomir_app_v1.md
    └── Договор_ethnomir_app_v1.docx
```

## Правила работы с веткой

- Репозиторий **приватный** — доступ к ветке имеют только члены организации `billionsx`.
- Каждая новая версия документа сохраняется рядом с предыдущей с суффиксом `_v2`, `_v3` и т. д., чтобы история правок была видна и в Git, и в именах файлов.
- После подписания финальной версии — в этой же папке появляется PDF с подписями и печатями (например, `Договор_ethnomir_app_final_signed.pdf`).
- В ветку **не заливается** код приложения, миграции, секреты, персональные данные гостей.
