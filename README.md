## Как возобновить работу в новом чате

1. **Клонируй репозиторий** в рабочее окружение: `git clone https://github.com/billionsx/ethnomir-spravochnik-v2.git`
2. **Прочитай** `docs/PROJECT-NOTES.md` — там зафиксированы все технические решения, баги, их причины и фиксы
3. **Рабочие пути в коде** — `src/pdfkit.py` содержит константы `FONT_DIR = "/home/claude/ethnomir/fonts/ttf"` и `SCREENS_DIR = "/home/claude/ethnomir/screens-all"`. При работе в новой среде — перенаправь на соответствующие папки из этого репо
4. **Для любых изменений** — редактируй `src/sections_*.py`, пересобирай через `build_full.py`, коммить в этот репо
