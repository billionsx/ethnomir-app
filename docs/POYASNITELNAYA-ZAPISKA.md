# Справочник продукта ethnomir.app · v2.0 · Апрель 2026

## Что это

Финальная версия справочника — **39 страниц А4**, 69 снимков интерфейса, 9 МБ PDF. Документ подготовлен для основателя и руководства парка Этномир. Описывает архитектуру, модули, CRM и ценностные карты по аудиториям на реальных данных production-системы.

## Где что лежит

- **PDF:** `pdf/ethnomir_spravochnik_v2.pdf` (39 стр, ~9 МБ)
- **Исходники генерации (Python + ReportLab):**
  - `src/pdfkit.py` — дизайн-система, палитра, шрифты, `draw_screen`, таблицы, утилиты
  - `src/sections_01_09.py` — обложка, оглавление, секции I (Продукт, Команда) и II (Архитектура) — стр. 1-12
  - `src/sections_III.py` — модули M00-M14 — стр. 13-28
  - `src/sections_IV.py` — CRM (Режим Владелец) — стр. 29-34
  - `src/sections_V_VI.py` — Ценностные карты — стр. 35-39
  - `src/build_full.py` — главный runner, собирает весь PDF
- **Скрины приложения:** `screens/` — 69 PNG 784×2024 px (уже пропущены через PIL `convert('RGB')`, без ICC-профилей)
- **Предпросмотр страниц:** `preview/pg-01.png` … `preview/pg-39.png` (110 DPI)
- **Шрифты:** `assets/fonts/Inter-*.ttf` (400, 500, 600, 700, 800, 900)
- **Предыдущая версия (для сравнения):** `source-pdf/ethnomir-spravochnik-v1-44pp.pdf`

## Как собрать

```bash
cd /путь/к/репо
python3 src/build_full.py
# PDF генерируется в pdf/ethnomir_spravochnik_v2.pdf
```

Требования: Python 3.10+, `reportlab`, `pillow`. См. `requirements.txt`.

## Ключевые особенности версии

1. **Яркость скринов.** Был баг ReportLab: `c.clipPath()` накапливал graphics state, второй и третий скрин на странице получались блёклыми. Решение: `clipPath` убран, скругление углов делается в PIL (RGBA-маска композитится на белом фоне), скрин отдаётся как in-memory JPEG q=92 через ImageReader. Коммит `662f6c1`.

2. **Страница «Команда» (стр. 6).** McKinsey one-pager с четырьмя партнёрами Billions X, hairline-дивайдерами, номерами 01-04, именами Inter-Bold 20pt, ролями Inter-Med 10.5pt, направлениями Inter 10pt. Два абзаца лида в стиле Forbes. Партнёр 04 скрыт: `***** ******` с подписью «(из-за вопросов конфиденциальности)», роль «Главный технический архитектор баз данных и ИИ приложения Этномира», описание «Действующий ведущий дата-инженер платформы данных и ИИ в Booking.com, Нидерланды.»

3. **120+ экранов в приложении.** Стр. 4, KPI «120+ экранов» — подсчёт: 5 главных табов + 21 подтаб + 29 CRM вкладок + 16 Паспорт views + 22 лендинга + 17 онбординг + 8 модалок + 2 вспомогательных.

4. **22 лендинга в `landing_pages` Supabase.** Полный список (slug · title_ru):
   - Бизнес: `franchise`, `realty`, `arenda`, `business`, `build`, `corp`, `travel` (7)
   - Этномир-смысл: `founder`, `heritage`, `vision2030`, `ethnomir3000`, `charity`, `recycling` (6)
   - Для гостей: `gift`, `educ`, `farm`, `jobs`, `directions`, `reviews` (6)
   - Контент: `articles`, `faq`, `agreement` (3)

5. **Типографика.** Inter 400/500/600/700/800/900. Заголовки Inter-Ex 34-36pt. Подзаголовки Inter-Med 13-14pt. Текст Inter 10-10.5pt leading 14-15.

6. **Цветовая схема.** Apple iOS 26+ Liquid Glass: чёрный #000 для label, #8E8E93 для label2, цвета секций (sec_I голубой, sec_II синий, sec_III зелёный, sec_IV чёрный, sec_V оранжевый). Палитра в `pdfkit.py` словарь `C`.
