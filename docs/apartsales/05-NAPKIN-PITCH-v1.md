## 2. Главное изобретение — Apartsales Lock

То, чего нет ни у ЦИАН, ни у Домклика, ни у Авито, ни у одного агентства в РФ:

### Гибрид 1+3 = Юридический + Поведенческий слой

**Юридический слой:**

- Агентский договор по ст. 1005 ГК РФ
- SMS-верификация лида в CRM застройщика **до** первого контакта
- Штраф 200% за нарушение фиксации
- Лид невозможно «увести налево»

**Поведенческий слой:**

- Покупатель получает **PDF-сертификат на скидку 1%**
- Сертификат активен только если он назовёт свой Apartsales-код в шоу-руме
- У клиента **прямой денежный интерес** защищать нашу фиксацию
- Покупатель становится защитником транзакции, а не объектом перехвата

### Эффект

Это решает главную боль рынка: 30–50% утечки лидов в обычных лид-моделях. Никто другой эту проблему не закрыл — это наш core IP.

| Игрок | Что они делают | Что у них **сломано** | Что у нас работает |

| **Агентство недвижимости** | Человек-агент водит клиента | Дорого (5–10% сделки), медленно, человеческий фактор, оффлайн | AI-консьерж = 80% работы агента, 0 офисов, 3% комиссия |
| **Отдел продаж застройщика** | Закрытый прайс одного ЖК | Один застройщик = нет сравнения, конфликт интересов | Нейтральный сравнитель, **несравнимый** ценностный слой для покупателя |

| **ЦИАН** | Каталог + продажа лидов CPL | Generalist, UX 2015 года, лиды утекают (нет Lock'а), застройщики платят за пустые лиды | Specialist, AI-first, **закрытая комиссия только за сделку**, Lead Lock |

| **Новостройкино** | Тематический каталог новостроек | Нет AI, нет Lock'а, узкий scale | AI + Lock + Yield + полный цикл сделки |
| **Domoos.Leads** | B2B-биржа лидов | Не B2C-бренд, продают лиды поштучно | Сильный B2C-бренд + ICP-фокус (московский инвестор-курортник) |

## 5. 7 одновременно — структурное преимущество

Структурно мы **единственные**, у кого есть **все 7 одновременно**:

1. **AI-консьерж** как первичный UX (не каталог)
2. **Lead Lock** с юридической + поведенческой защитой
3. **Сертификат покупателю** (1% обратно — Aviasales-эффект)
4. **Yield-прогноз и Price Forecast** через ML (точность ±15%)
5. **Полный цикл сделки в приложении** через Госключ + ЕСИА + СБП + Росреестр
6. **Apartsales Yield** — управление сдачей в аренду удалённо для приезжих инвесторов (8–12% маржа на rent-roll)
7. **Asset-light dispatch** — ноль офисов, ноль штатных агентов, всё через CRM застройщика

У каждого конкурента есть 1–3 из этих 7. Ни у кого нет всех 7.

## 6. Почему моат держится

**Категориальный pivot.** Мы не «ещё одно агентство» и не «ещё один портал». Мы **новая категория** — метапоиск с фиксационным движком. Конкурентам надо менять архитектуру, культуру и контракты — это 18–24 месяца минимум.

**Network effect через Lock.** Чем больше застройщиков подписывают эксклюзивные lock-договоры — тем сложнее ЦИАНу/Самолёту убедить их разорвать.

**Data moat.** Через 12 месяцев у нас будет уникальный датасет по реальным yield'ам Крыма, которого нет ни у Сбера, ни у ЦИАНа.

**Покупатель в нашей команде.** Сертификат превращает каждого клиента в защитника фиксации — это **поведенческий моат**, а не только юридический.

**Specialty-focus vs Generalist.** Все крупные конкуренты — generalist'ы (вся РФ, все сегменты). Мы — focused specialist на курортных инвестициях. В 90% случаев focused бьёт generalist'а.

## 7. Одной фразой для лифта

> **Apartsales забирает у агентств их работу (через AI), у Авито/ЦИАН — их лиды (через Lock), у банков — их транзакции (через нейтральный API), у Тинькофф — их философию (ноль офисов), у Aviasales — их модель (% с закрытой сделки), и отдаёт покупателю 1% обратно.**
