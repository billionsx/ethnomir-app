## #9 — Open API & Webhook Layer

**Что:** Публичный, документированный API для двусторонней синхронизации с любой CRM застройщика.

**Поддерживаемые интеграции из коробки:**
- Bitrix24
- AmoCRM
- Profitbase
- 1С Управление недвижимостью
- Битрикс24.Облако
- HubSpot
- Salesforce (для крупных)
- Zoho
- кастомные через Open API

**Возможности API:**
- CRUD юнитов и комплексов.
- Синхронизация цен (двусторонняя).
- Webhook на event lead-created → попадает в их CRM.
- Webhook на event price-changed → синхронизируется на нашу платформу.
- KYC-результаты передаются в их систему.
- Документы сделки доступны через API.

**Стандарты:** OpenAPI 3.1, REST + опциональный GraphQL, OAuth 2.0, idempotency keys, rate limits.

**Lock-in:** Кто интегрировался — тот не уйдёт. Это **технический моат**.

## #10 — White-label CRM для «Excel-застройщиков»

**Целевой клиент:** Застройщик с <100 юнитов в год, лиды в Excel, нет CRM, нет сайта.

**Что даём:**
1. **Сайт застройщика** на нашем CMS (поддомен или их домен).
2. **CRM воронка** — лиды, статусы, сделки, отчётность.
3. **Календарь броней / показов**.
4. **AI-ассистент** менеджеру (генерация писем, follow-up).
5. **Reporting** — разрезы по юнитам, менеджерам, источникам.
6. **Биллинг** — выставление счетов, договоры.
7. **Электронный документооборот** через ЭЦП.

**Pricing:**
- Free tier — до 20 юнитов.
- Pro — от 4900 ₽/мес, включает все модули.
- Enterprise — от 49900 ₽/мес для среднего сегмента.

**Lock-in максимальный:** Мы становимся их единственным IT-стеком. Уйти = переехать всю компанию.

**Решает проблему:** «Застройщики собирают лиды в Excel, лиды теряются». Мы устраняем эту фрикцию полностью.

## #11 — SLA-движок

**Что:** Автоматическая система контроля скорости ответа застройщика клиенту.

**Правила:**
- Лид пришёл → у застройщика **2 часа** на первый контакт (рабочее время).
- Через 2 часа без контакта — лид **переадресуется** второму застройщику в той же категории.
- Каждый «потерянный» лид снижает Developer Score → застройщик проседает в выдаче (#1).
- 3 пропущенных лида подряд = временное исключение из топ-выдачи.

**Метрики, видимые клиенту:**
- «Этот застройщик отвечает в среднем за 14 минут.»
- «98% лидов получают ответ в течение часа.»

**Зачем:**
- Защита клиента (никто не висит без ответа).
- Стандарт индустрии (мы устанавливаем планку).
- Конкуренция между застройщиками за быстрый ответ → продукт улучшается.

**Что:** Обучение и сертификация маркетологов / риэлторов / менеджеров застройщиков.

**Формат:**
- Онлайн-курсы (10–80 часов).
- Сертификации Apartsales Certified Sales Manager / CSM Pro / Master.
- Партнёрство со Сколково / РЭШ / NYU SPS / INSEAD для престижа.
- Локальные офлайн-конференции в РФ / ОАЭ / Турции.

**Программы:**
1. Apartsales for Developers — основы работы с платформой.
2. Cross-Border Sales — для тех, кто продаёт иностранцам.
3. AI Tools for Real Estate Agents — концепции AI Concierge / Pricing.
4. ESG and Carbon for Developers — для соответствия #36.
5. Apartsales Master Program — для топ-менеджеров застройщиков.

**Зачем:**
- Создаёт **сообщество и стандарты индустрии**.
- Превращает Apartsales в **институт**, а не маркетплейс.
- Догнать институт — невозможно. Это brand moat.

**Revenue:** Подписки + сертификационные сборы + партнёрские курсы.

## #30 — Apartsales Studio (AI-Generated Buildings)

**Что:** Мы анализируем спрос через свои данные → проектируем эталонные концепции ЖК → продаём как франшизу/лицензию застройщикам.

**Логика:**
- Из миллионов поисков и сделок мы видим: «в Сочи дефицит 1BR с видом на море до 8M ₽» или «в Дубае не хватает walk-up villas с зелёным двором».
- Apartsales Studio проектирует эталонный концепт ЖК под этот спрос.
- Продаём концепт-пакет (3D-проект + спецификации + материалы + ESG-сертификация) застройщикам.
- Опция: включаем своё имя в название ЖК (Apartsales-certified concept) → premium-цена.

**Зачем:** Мы становимся **upstream от застройщика**. Не просто витрина — **генератор того, что строится**.

**Pricing:** Лицензия концепта 5–50M ₽ за проект + роялти.

## Стандартизация процессов

(Связь со столпом 6 из `02-product-pillars.md`.)

**Стандартные документы каждого юнита:**
- ДДУ / эквивалент в иностранной юрисдикции.
- Технический паспорт.
- Разрешение на строительство.
- Проектная декларация.
- Финансовая отчётность застройщика.
- Страховка титула (#41).

Все автоматически валидируются + сторятся в Time Machine (#40).

**Стандартный онбординг застройщика:**
- Шаг 1: регистрация юрлица (ЭЦП-валидация автоматическая).
- Шаг 2: загрузка лицензий (OCR + AI-валидация).
- Шаг 3: подключение CRM или регистрация на White-label.
- Шаг 4: загрузка первых ЖК и юнитов.
- Шаг 5: подписание мастер-соглашения.
- **Время: 1 рабочий день.**
