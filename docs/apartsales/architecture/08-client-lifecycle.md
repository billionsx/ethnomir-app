# 08. Жизненный цикл клиента

## Жизненный цикл

```
Discover → Match → KYC → Tour → Book → Close → Live → Manage → Resell → Inherit
   ↑                                              ↓
   └──────────────────── Loyalty Loop ────────────┘
```

Каждая фаза — отдельный продукт.

## #13 — УК-as-a-Service (после сделки)

**Что:** Управление недвижимостью клиента как сервис.

**Целевые клиенты:** P2 (инвесторы), P3 (курортники), P5 (иностранцы).

**Услуги:**
- Сдача в долгосрочную / краткосрочную аренду (Booking / Airbnb / Циан / локальные).
- Обслуживание (уборка, ремонт, управление коммуналкой).
- Налоговая отчётность.
- Страховка.
- Регулярные отчёты владельцу через приложение.

**Pricing модель:**
- 10–25% от арендного дохода.
- Либо flat fee 3000–15000 ₽/мес.
- Либо «хочу хедж по доходу» — гарантированный yield 5–7%/год, мы берём risk.

**Зачем:**
- Превращает разовую сделку в **подписку**.
- LTV клиента вырастает в 5–10 раз.
- Данные о реальном yield → улучшают AS-Index и Score™.

**Roadmap:** Y1 — пилот в Сочи (50 объектов). Y2 — масштаб на 5 регионов. Y3 — собственные УК-партнёрства в каждом регионе экспансии.

## #14 — Resale / Secondary Loop

**Что:** Через 3–7 лет тот же юнит выходит на ресейл через нас.

**Архитектурное решение:**
- Каждый юнит проданный через нас — **навсегда** в нашей базе.
- При желании владельца продать — Apartsales включается с приоритетом.
- История юнита (Apartsales Time Machine #40) → доверие нового покупателя.
- Оценка цены через Quantum Pricing Engine (#48).
- Опция мгновенной ликвидности через Real Estate Exchange (#34) или Emergency Resale Protection (#45).

**Зачем:** Без resale loop клиент **мёртв после первой покупки**. Это блокирует LTV. Resale loop = повторная комиссия + удержание клиента.

**Программы (топ-10 по mass-market):**

3. Грузия — ВНЖ через инвестицию $100K.
4. Кипр — Permanent Residency через €300K.
5. Португалия — Golden Visa.
6. Мальта — гражданство через программу.
7. Греция — Golden Visa.
8. Узбекистан — упрощённое ВНЖ для инвесторов.
9. Сербия — упрощённое получение ПМЖ.
10. РФ — для иностранцев через ВНЖ + инвестиции.

**Услуги:**
- Подбор программы под цели и бюджет.
- Юр. сопровождение (партнёрство с миграционными бюро в каждой стране).
- Подготовка документов.
- Сопровождение в посольстве / консульстве.
- Tax engine для налоговой оптимизации после переезда (#44).

**Pricing:** 5–20% от стоимости визы / ВНЖ + bundled со сделкой.

## #32 — Generational Residency Bundle

**Что:** Пакет на всю семью — «квартира + ВНЖ + школа детям + банк + страховка + перевозка» в один клик.

**Целевые клиенты:** P5, P7, P8, P10.

**Пример пакета «Russia → UAE»:**
- Квартира 2BR в Дубае (от $350K).
- Golden Visa (10 лет) для семьи.
- Места в IB-школе (через партнёрство).
- Открытие счёта в Mashreq / FAB через KYC-pass.
- Health insurance на семью.
- Перевозка вещей через партнёра.
- Tax-консультация по выходу из РФ + входу в ОАЭ.

**Pricing:** Bundle markup 10–15% over individual services + комиссия за квартиру.

**Размер рынка:** Россияне в ОАЭ — 100K+ переехали с 2022. Турки в РФ — 50K+. Иранцы в Турции — 200K+. Это **миллионный рынок**.

**Что:** Private banking-уровень для покупателей от $5M.

**Состав:**
- Личный менеджер 24/7 (one-to-one).
- Доступ к off-market сделкам (юниты, не выставленные публично).
- Быстрый KYC (24 часа).
- Частные показы / приватные туры.
- Доступ к private events (форумы инвесторов, семинары с топ-девелоперами).
- Apartsales Constitution-уровень юр. защита (#46).
- Прямая линия в Apartsales Bank (#24) — спец-условия.

**Pricing:** Annual fee $25K–250K в зависимости от tier.

**Маржа:** 3–5% per deal vs 1.5% standard. LTV в десятки раз выше.

## #53 — Great Wealth Transfer Product

**Что:** Продукты для наследования недвижимости. $84T перейдёт от бумеров к их детям до 2045 — крупнейший трансфер капитала в истории.

**Продукты:**

1. **Smart-завещания на блокчейне.**
   - Условия наследования закодированы в смарт-контракте.
   - При триггере (death certificate via API регистратора) → автоматический transfer NFT-Title (#39).
   - Пример: «Эта квартира → дочь, при условии 18+ возраста и 2 лет учёбы».

2. **Цифровые совместные собственники.**
   - Доли семьи как токены, можно перераспределять.
   - Голосование о продаже / сдаче в аренду через DAO-механизмы.

3. **Постхумусный transfer NFT-Title.**
   - Связан со смерт-сертификатами через API.
   - Уберёт юр. бой между наследниками.

4. **Трасты для внуков.**
   - Apartsales Trust — юр. структура для multi-generational ownership.
   - Распределение дохода от УК-as-Service автоматически по бенефициарам.

5. **Inheritance Tax Engine.**
   - Расчёт налога на наследство в каждой юрисдикции.
   - Стратегии оптимизации (трасты, dual-citizenship, family offices).

**Зачем:** Никто не позиционирован на этой нише в недвижимости. Огромный blue ocean.
