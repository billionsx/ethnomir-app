# 13. Roadmap и география экспансии

## Принцип фазирования

> 5-летний горизонт. Каждая фаза имеет одну главную ставку и 2–3 поддерживающие.

```
PHASE 0: Foundation       (M0–M3)    Спецификация + дизайн + MVP-скелет
PHASE 1: Beachhead         (M3–M12)  MVP + 1 регион + 50 застройщиков
PHASE 2: Regional Scale    (M12–M24) Multi-region + AS-Index + B2B2C
PHASE 3: Global Player     (M24–M48) Bank + Exchange + Sovereign
PHASE 4: IPO + Adjacency   (M48–M60) Triple listing + luxury assets
```

## Ключевые риски roadmap

1. **Регуляторика** — задержки лицензий банка / страхования / NFT-Title. Митигация: параллельные заявки в нескольких юрисдикциях.

2. **Капитал** — недостаток финансирования для роста. Митигация: SWF partnerships (#61), revenue from operations к Y2.

3. **Talent** — невозможность найма топ-50 проп-тех людей. Митигация: equity-heavy compensation + ремоут-команды.

4. **Геополитика** — санкции / войны блокируют cross-border. Митигация: multi-region structure, SWF из 5 стран как геополитический щит.

5. **Конкурент-копирование** — См. `10-defensibility-moat.md`.
