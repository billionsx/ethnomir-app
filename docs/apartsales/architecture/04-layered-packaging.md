# 04. Слоёная упаковка — победа на каждом слое

## 1. Тезис

Конкуренты продают **юниты**. Apartsales продаёт **слои ценности**:

```
СТРАНА → РЕГИОН → ГОРОД → ЗАСТРОЙЩИК → КОМПЛЕКС → ЮНИТ
```

На каждом слое — отдельный landing, отдельный SEO, отдельный AI Concierge prompt, отдельный набор UGC-контента. Покупатель приходит на любом слое и идёт вглубь воронки.

**Архитектурно** это реализовано через **Headless Content Layer (#20)**: одна headless CMS, шаблоны под каждый тип слоя, автогенерация миллионов страниц на 6 языках.

## 3. Шаблоны контента (headless CMS)

### Принцип

Один шаблон = миллионы страниц. Контент **автогенерируется** из структурированных данных + AI-обогащение.

### 6 типов шаблонов

| Тип шаблона      | Слой       | Уникальные блоки                          |
|------------------|------------|-------------------------------------------|
| `country-template` | СТРАНА   | Visa, taxation, big-numbers, regions      |
| `region-template`  | РЕГИОН   | Subregions, climate, demographics         |
| `city-template`    | ГОРОД    | Districts, infrastructure, transport      |
| `dev-template`     | ЗАСТРОЙЩИК | Score, projects, financials             |
| `project-template` | КОМПЛЕКС | Twin, infrastructure, DAO, ESG            |
| `unit-template`    | ЮНИТ     | Tour, pricing, mortgage, concierge        |

### Языки

Каждая страница автоматически генерится на **6 языках**: RU / EN / AR / CN / TR / ES. AI-перевод + редактура локальных экспертов (#18) для топ-10K страниц.

### SEO

- Уникальный title, meta-description, schema.org для каждой.
- Internal linking автоматический (страна ↔ регион ↔ город ↔ ЖК).
- Sitemap auto-generation, ping Google/Yandex/Baidu/Yandex.

## 4. Уникальный SEO-моат

**Объём:**
- 50 стран × 500 регионов × 5000 городов × 50000 застройщиков × 200000 ЖК × 5M юнитов = **5M+ страниц на 6 языках = 30M+ страниц контента**.
- Через 2 года все они проиндексированы → доминантная SEO-позиция.
- Конкурент скопировать: 18–24 месяца минимум на построение CMS-инфраструктуры + ещё 12 на индексацию.

**Это и есть слоёный SEO-моат.**

## 5. Маркетинговая упаковка как продукт для застройщика

Каждому застройщику мы даём **профессиональный коммерческий блок**:

- ★ Шаблонный landing ЖК на нашем CMS (10 минут на запуск).
- ★ Профессиональная фотосессия + 3D-тур + drone-видео (по подписке).
- ★ AI-копирайтинг описаний (генерация и переводы).
- ★ Co-branded реклама в нашей Media Empire (#58).
- ★ Лендинги под рекламные кампании (Google Ads / Я.Директ / Meta).

Это решает идею «профессионального уровня упаковки на высшем уровне» — мы делаем за застройщика то, что он сам сделать не может (или сделает плохо).
